# 自动化工资条邮件发送系统

一个基于 Python 的企业级工资条批量发送工具，支持 Excel 读取、多线程并发、失败自动重试、实时进度显示。

## ✨ 功能特性

- 📊 **Excel 数据驱动**：自动读取员工工资信息
- 🚀 **多线程并发**：支持可控并发数，大幅提升发送效率
- 🔄 **失败自动重试**：每封邮件失败自动重试，提高成功率
- 📈 **实时进度监控**：控制台实时显示成功/失败/剩余数量
- 🛡️ **线程安全**：每个线程独立 SMTP 连接，无资源冲突
- 🎨 **精美 HTML 模板**：自带专业的工资条表格样式
